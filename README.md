# 进度
* http://localhost:3000/ 首页位置，放宣传文档
* http://localhost:3000/ 测试页，展示各个框架的可用情况
* http://localhost:3000/login 用户登陆页，首页的开始学习按钮引导到登陆页
* http://localhost:3000/dashboard 用户登陆以后的后台中心

## 技术栈
* SQLite - 浏览器端 SQL 数据库 (sql.js)
* LokiJS - 浏览器端文档型数据库 (类似 MongoDB)
* Prisma - Prisma 风格的 ORM API (浏览器端实现)
* Express - Node.js 后端服务器
* d3.js - 数据图表展示框架
* Vue 3 - 前端框架
