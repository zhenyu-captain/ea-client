## 技术栈

* SQLite - 浏览器端 SQL 数据库 (sql.js)
* LokiJS - 浏览器端文档型数据库 (类似 MongoDB)
* Prisma - Prisma 风格的 ORM API (浏览器端实现)
* Express - Node.js 后端服务器
* Vue 3 - 前端框架

## 测试
* npm install
* npm run dev:all
* zsh -c "source ~/.zshrc && node server/check-db.js"
* http://localhost:3000/test
